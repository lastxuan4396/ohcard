# OH Card Studio（动态网页版）

## 你会得到什么
- 多玩法融合：单卡聚焦、时间线三张、关系镜像、四维钻石、五点策略阵
- 抽卡动画：牌堆洗牌 + 卡牌翻转 + 分步出现
- 引导提问：描述性 / 联想性 / 探究性三层问题引导
- 功能增强：专注计时、本地历史、回看、删除、导出引导文本、笔记保存
- 响应式布局：桌面与手机均可使用

## 运行方式
1. 直接用浏览器打开 `index.html`
2. 或在目录启动静态服务（可选）：
   - `python3 -m http.server 8080`
   - 浏览器访问 `http://localhost:8080`

## 文件说明
- `index.html`: 页面结构
- `styles.css`: 页面视觉与动画
- `script.js`: 抽卡与引导逻辑
